Docker network to connecting to docker containers together
creating network for multiple container:

$ docker network create <network-name>
$ docker network ls

NB!! docker-compose creates network by default. The commands above are not needed.

::::: database.yml   :::::
$ docker-compose database.yml -f up #starting containers and conting them

$ docker-compose -f database.yml down #stoping the containers
